# 🚀 AI Platform

A premium AI SaaS landing page built with React and Vite, featuring a modern UI, enterprise-inspired design, responsive layouts, pricing plans, interactive components, and Google Form integration.

## 🌐 Live Demo

https://ai-platform-final.vercel.app/

## 📂 GitHub Repository

https://github.com/anushka17jaiswal/ai-platform-final

---

## ✨ Features

- 🎨 Modern SaaS Landing Page
- 📱 Fully Responsive Design
- 🤖 AI Workflow Showcase
- 📊 Feature Cards with SVG Assets
- 💰 Interactive Pricing Section
- 🌍 Multi-Currency Pricing
- 📅 Monthly / Yearly Billing Toggle
- ❓ FAQ Accordion
- 📨 Google Form + Google Sheets Integration
- ⚡ SEO Optimized
- 🎯 Smooth UI & Hover Animations

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/anushka17jaiswal/ai-platform-final.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
